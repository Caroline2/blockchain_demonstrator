A simple user manual

May 24, 2021

It is highly recommended to watch the introduction video and static info form homepage before using the app by yourself.

You were able to find this article, which means you were able to run the app and go through the homepage. That's a good start! Hope you enjoyed reading some static info about blockchain. But now let's learn about some more dynamic features!

Reading page

Now you're on the reading page. From here you can be simply redirected back to homepage, to the second article (scroll down or click here) or to the login page.

Login & Profile page

The login page enables you to create an account, then log in and use more features.

You can display almost all the pages even if you aren't logged in (now you can try for instance this or this), but aren't able to make post requests.

So the first step is simple – click Login, create an account and log in. Then you will be redirected to the profile page, where you can find some info about yourself:

When you are logged in, go back to homepage and click Login, you are automatically redirected to the profile page.

Blockchain page

On the blockchain page the final blockchain is displayed. Nice, isn't it?

New transaction page

On this page you can send a simple transaction to any other existing user. Input is very intuitive, just don't forget to specify which coins you are sending. If you make a mistage, a flash message will occure and tell you, what have you made wrong.

Blockchain page

On the blockchain page the final blockchain is displayed. Nice, isn't it?

Mine page

Here the pending transactions are displayed. If there is at least five of them, you can click Mine these transactions button and mine a new block. Awesome!

Notice the differences between different types of transaction listing. Can you figure out the rules without checking in the SOC file (yet just in Czech, I'm sorry)?

After adding a new block to the block, don't forget to check the updated Blockchain page!

Sell page

You can also send some of your electricity coins (by default you get 10 of them after registration). The mechanism is simmilar to sending a simple transaction – input the amount of electricity tokens you want to sell, input the amount of money tokens you want to receive and click Create an offer button.

Changed your mind with some offer? You can delete in on the profile page. Just scroll down, click Delete some offer, input the number of the offer you want to delete and click the Delete button.

Buy page

The buy page is the place where the offers are displayed. Click the Buy button, input the number of the offer you want to take advantage of and click the Buy button again.

27 August 2021 update: Some of the offer issues have been solved. Some features still aren't 100% functional.

Logout page

This doesn't need any special explanation, I guess – just click Logout and you'll see what happens.

That's it! Now you're fully able to use the app. Enjoy!
